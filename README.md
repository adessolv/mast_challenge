# 6 Week Pleasure Challenge

React Native / Expo приложение для приватного 6-недельного челленджа. Пользователь отмечает активные дни в календаре, фиксирует длительность практики, ставит оценку результата по 5-балльной шкале и добавляет заметку.

## Запуск

```bash
npm install
npm run start
```

Далее откройте проект в Expo Go, Android Emulator или iOS Simulator.

## Assets

В репозитории намеренно нет бинарных изображений, чтобы PR можно было создать в окружениях, где бинарные файлы не поддерживаются. Expo будет использовать стандартные placeholder-иконки, пока вы не добавите собственные ассеты локально.
